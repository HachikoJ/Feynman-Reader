# 费曼阅读法 | Feynman Reading Method

用费曼学习法深度理解一本书的 Web 应用。

## 功能特点

- 🔍 **背景探索** - 了解作者生平、时代背景、写作动机
- 📚 **全书概览** - 建立整体认知框架
- 🎯 **深度拆解** - 以教代学，找出理解盲点
- ⚖️ **辩证分析** - 批判性思考，寻找漏洞和反面观点
- 🌍 **众声回响** - 后世评价与历史影响
- 🔗 **融会贯通** - 与已有知识体系建立连接

## 技术栈

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- DeepSeek API

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

打开 http://localhost:3000，配置你的 DeepSeek API Key，输入书名即可开始。

## 获取 DeepSeek API Key

访问 https://platform.deepseek.com/api_keys 注册并获取 API Key。
