# 费曼实践提醒优化

## 优化目标
将费曼实践下的完成提醒做得更加醒目，让用户清楚知道完成阅读的条件。

## 优化前后对比

### 优化前
```
简单的黄色背景框
小字体提示
单行文字
不够醒目
```

### 优化后
```
渐变背景（黄→橙→红）
大图标 + 大标题
清单式展示
实时状态反馈
动画效果
```

## 新设计特点

### 1. 视觉层次
- **大图标**：5xl尺寸的⚠️警告图标，带慢速弹跳动画
- **大标题**：xl尺寸的"📋 完成阅读条件"
- **清单式**：两个条件分别列出，一目了然
- **状态指示**：圆形数字/对勾，实时显示完成状态

### 2. 颜色方案
- **渐变背景**：黄色→橙色→红色，营造紧迫感
- **双层边框**：2px黄色边框，更加醒目
- **阴影效果**：shadow-lg，增加立体感
- **文字颜色**：
  - 标题：text-yellow-300（亮黄色）
  - 条件：text-yellow-200（柔和黄色）
  - 提示：text-yellow-300/80（半透明）

### 3. 动画效果
- **整体脉动**：animate-pulse-slow（3秒循环）
- **图标弹跳**：animate-bounce-slow（2秒循环）
- **吸引注意**：不会太快，不会太慢，刚刚好

### 4. 内容结构
```
┌─────────────────────────────────────────┐
│ ⚠️  📋 完成阅读条件                      │
│                                         │
│  ① 教学模拟达到 60 分以上               │
│  ② 角色问答全部达到 60 分以上           │
│                                         │
│  💡 完成以上两项后，本书将自动归档到"已读" │
└─────────────────────────────────────────┘
```

### 5. 实时状态反馈
- **未完成**：灰色圆圈 + 数字（1、2）
- **已完成**：绿色圆圈 + 白色对勾（✓）
- **动态更新**：完成条件后立即变绿

### 2. CSS动画
```css
/* 慢速脉动 - 整体透明度变化 */
@keyframes pulse-slow {
  0%, 100% { opacity: 1; }
  50% { opacity: 0.85; }
}
.animate-pulse-slow {
  animation: pulse-slow 3s ease-in-out infinite;
}

/* 慢速弹跳 - 图标上下移动 */
@keyframes bounce-slow {
  0%, 100% { transform: translateY(0); }
  50% { transform: translateY(-5px); }
}
.animate-bounce-slow {
  animation: bounce-slow 2s ease-in-out infinite;
}
```

### 3. 状态判断逻辑
```typescript
// 教学模拟是否通过
const teachingPassed = practiceRecords.some(r => r.scores.overall >= 60)

// 角色问答是否通过
const qaPassed = book.qaPracticeRecords && 
  book.qaPracticeRecords.length > 0 && 
  book.qaPracticeRecords[book.qaPracticeRecords.length - 1].allPassed
```
